# MMR-Prog-Colon：An Interpretable Multimodal Multitask Transformer Framework for Joint Predicting MMR Status and Prognostic Outcomes in Radically Resected Stage Ⅱ-Ⅲ Colon Cancer: A Multicenter Study

## Abstract

Background: Conventional TNM staging system fails to adequately characterize tumor heterogeneity, limiting accurate prognostic prediction in stage Ⅱ-Ⅲ colon cancer (CC). We developed MMR-Prog-Colon, a Transformer-based multimodal multitask deep learning framework that integrates clinical, radiomic, and pathological data to jointly predict disease-free survival (DFS), overall survival (OS), and mismatch repair (MMR) status—thus providing a more precise, interpretable artificial intelligence (AI) tool for individualized clinical decision-making.
Materials and Methods: This multicenter retrospective study enrolled 907 radically resected stage Ⅱ–Ⅲ CC patients from four medical centers (median follow-up, 75.1 months). Patients were divided into a training set (n = 419), an internal validation set (n = 180), and two independent external test sets (n = 236 and n = 72, respectively). DFS was the primary task, with OS and MMR status as secondary tasks. Cox regression analyses were conducted to identify clinical features associated with DFS. From preoperative venous phase CT images, we compared conventional, habitat, and multi-scale peritumoral radiomics (1 mm, 3 mm and 5 mm concentric annular regions without mutual overlap) to identify the optimal feature strategy. For postoperative hematoxylin-eosin (H&E)-stained whole-slide images (WSIs), we extracted 206-dimensional patient-level pathomics features via a three-step pipeline: WSI preprocessing, patch-level feature extraction, and multiple instance learning (MIL) aggregation. Based on the above multi-source heterogeneous data, we constructed a Transformer-based multimodal early fusion and multitask learning model. Model performance was assessed using the concordance index (C-index), Kaplan–Meier analysis, and time-dependent receiver operating characteristic (ROC) analysis, and compared against single-modal and late fusion stacking models. Hover-Net and quantitative immunohistochemistry (CD3, CD8, FOXP3, etc.) were used to explore the biological basis of model decisions.
Results: Age, pN stage, and perineural invasion (PNI) were independent prognostic factors for DFS. Habitat radiomics outperformed conventional radiomics for DFS prediction (C-index: 0.701 vs 0.523, in training set); the Peri3mm model showed the most stable generalization (1-year AUC = 0.779, 3-year AUC = 0.793 in external test set 2). MMR-Prog-Colon achieved robust DFS prediction across cohorts (C-index: 0.812–0.884), outperforming the best single-modal model (ΔC-index = 0.088, in external test set 1) and the late fusion stacking approach (ΔC-index = 0.024, in external test set 1; ΔC-index = 0.053, in external test set 2 ), with minimal cross-center variation (ΔC-index = 0.072). It also performed well in OS (C-index: 0.815–0.918) and MMR status prediction (AUC: 0.849–0.997). Gradient-weighted class activation mapping (Grad-CAM) visualization confirmed that the model decisions focused on key tumor regions with nuclear pleomorphism. High-risk group (low DFS score) patients exhibited an immunosuppressive microenvironment characterized by elevated tumor cellularity, CD3+/CD8+ T cell exhaustion, and Treg enrichment (P < 0.01), confirming biological plausibility.
Conclusion: To our knowledge, MMR-Prog-Colon is the first robust, interpretable multimodal multitask AI framework that integrates clinical, habitat radiomic, peritumoral radiomics and pathomic data via Transformer-based early fusion to jointly predict MMR status, DFS, and OS in stage Ⅱ–Ⅲ CC. In multicenter validation, it outperformed single-modal and late fusion stacking approaches; additionally, its biological plausibility (supported by qIHC and Hover-Net analyses) further confirms its clinical reliability and highlights strong potential for clinical translation.
