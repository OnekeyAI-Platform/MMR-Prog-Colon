# MMR-Prog-Colon: An Interpretable Multimodal Transformer Framework for Joint Prediction of MMR Status and Prognostic Outcomes in Radically Resected Stage II-III Colon Cancer

## Abstract

The conventional TNM staging system inadequately accounts for tumor heterogeneity, limiting prognostic accuracy in stage II–III colon cancer (CC). Here, we developed MMR-Prog-Colon, a transformer-based multimodal multitask framework integrating clinical, habitat/peri3mm radiomic, and pathological data to jointly predict disease-free survival (DFS), overall survival (OS), and mismatch repair (MMR) status. This multicenter retrospective study enrolled 907 patients with radically resected stage II–III CC from four centers (median follow-up: 75.1 months), divided into training (n=419), internal validation (n=180), and two external test sets (n=236, n=72). MMR-Prog-Colon achieved robust DFS prediction across cohorts (C-index: 0.812–0.884), outperforming the best single-modal model (ΔC-index = 0.088, in external test set 1) and the late fusion stacking approach (ΔC-index = 0.024/0.053, in two external test sets), with minimal cross-center variation (ΔC-index = 0.072). It also excelled in OS prediction (C-index: 0.815–0.918) and MMR status classification (AUC: 0.849–0.997). Analyses combining Hover-Net and quantitative immunohistochemistry revealed immunosuppressive microenvironments in high-risk patients. To our knowledge, MMR-Prog-Colon is the first interpretable multimodal multitask transformer framework for comprehensive prognostication in radically resected stage II–III CC, delivering superior performance, biological plausibility, and high translational potential.
